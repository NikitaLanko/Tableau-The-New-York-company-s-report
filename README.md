# *Analyzed a company out of New York, with the help of Tableau*:statue_of_liberty:	

## *Описание проекта*:statue_of_liberty:	
Этот проект представляет собой всесторонний анализ данных компании, расположенной в Нью-Йорке, с использованием Tableau. В проекте рассмотрены ключевые аспекты управления персоналом, такие как анализ распределения заработной платы, структура сотрудников по возрасту и полу, а также распределение сотрудников по штатам и должностям.

## *Используемые инструменты*:statue_of_liberty:	
Tableau: Для анализа и визуализации данных.
Источники данных: Таблицы с информацией о сотрудниках, заработных платах, возрастных группах, а также других данных.

## *Как использовать*:statue_of_liberty:	
Открытие файла: Откройте файл .twbx в Tableau.
Навигация: Переключайтесь между вкладками, чтобы просмотреть различные аспекты анализа.
Используйте фильтры для анализа данных по возрасту, полу, штату и другим параметрам.
